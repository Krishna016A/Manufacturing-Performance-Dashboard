# Manufacturing Performance Dashboard using Power BI

## 📌 Project Overview

This project is an interactive **Manufacturing Performance Dashboard** developed using **Power BI**. The dashboard provides a comprehensive view of key manufacturing metrics, enabling management to monitor performance across various departments and make data-driven decisions.

The report was built using manufacturing data from Excel and consists of multiple dashboards covering safety, production, productivity, inventory, quality, maintenance, and sales performance.

---

## 🎯 Objectives

* Monitor Actual vs Plan performance across key business areas.
* Identify trends and performance gaps.
* Track quality issues and corrective actions.
* Provide management with interactive reports for decision-making.

---
